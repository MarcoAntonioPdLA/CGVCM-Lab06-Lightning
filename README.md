# Iluminación

En el siguiente proyecto se creó la escena con una habitación, con puerta, ventana y diferentes decoraciones, como muebles, escritorio, silla, computadora, etc.
El objetivo es mostrar cambios de iluminación en la habitación que representan diferentes emociones, como comodidad, tristeza y miedo.

## Controles
- WASD - moverse
- Mouse - rotar
- 1 - Cambiar a iluminación de comodidad
- 2 - Cambiar a iluminación de tristeza
- 3 - Cambiar a iluminación de miedo
- Esc - salir
